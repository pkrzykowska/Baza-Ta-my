# Baza-Ta-my
Projekt obejmuje zaprojektowanie oraz implementację systemu informatycznego służącego do gromadzenia, przetwarzania i analizy danych pomiarowych pozyskiwanych z obwodnic taśmowych. System składa się z relacyjnej bazy danych oraz aplikacji okienkowej (GUI) umożliwiającej wygodną obsługę, wprowadzanie danych oraz generowanie raportów.
